Утилита для вывода каталога файлов из TRD-образа для ОС Sprinter DSS.

Запуск:
TRDLIST.EXE filename.trd

Для сборки необходимо установить утилиту osfmount - https://www.osforensics.com/tools/mount-disk-images.html и
прописать путь к ней в Path, а так же в переменную среды %SPRINTER_EMULATOR%
необходимо записать путь к эмулятору ZXMAK2, в папку которого по итогу сборки
будет скопирован образ дискеты с программой
